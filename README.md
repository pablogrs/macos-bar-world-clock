# macOS World Clock

A lightweight macOS menu bar application to keep track of multiple time zones at a glance.

## Features

- **Menu Bar Display**: Shows the current time for multiple locations directly in your macOS menu bar.
- **Flags Support**: Uses emojis to represent different locations for quick identification.
- **Local Time**: Option to always show your current local time.
- **Configurable**: Easily add or remove time zones through the configuration window.
- **Custom Names**: Give your time zones friendly names (e.g., "Home", "Office", "Teammate").

## Getting Started

### Prerequisites

- macOS 11.0 or later
- Xcode 12.0 or later (for building from source)

### Building

1. Open `hours.xcodeproj` in Xcode.
2. Select the `hours` scheme and your Mac as the destination.
3. Press `Cmd + R` to build and run.

## Usage

- Click the time display in the menu bar to see more details and access the configuration.
- Select **Configuration...** to manage your time zones.
- Use **Quit macOS World Clock** to exit the application.

## Configuration

In the Configuration window, you can:
- Toggle the display of your **Local Time**.
- Add new time zones from a searchable list.
- Remove time zones you no longer need.
- Rename time zones to better suit your needs.
